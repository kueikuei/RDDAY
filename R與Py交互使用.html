R 與 py 數據類型互轉¶

向量（Vector）是R的一個最重要的也是最常用的數據類型，可以理解為一個二維數據，對應Python的list。 在R控制台中，聲明一個變量：“x <- 1”，X會被聲明成一個向量，而其第一個值是1。 R常常用c()函數來聲創建一個由多個值組成的向量，例如c(1,2,3,4)。 Python要與R打交道，除了訪問R對象和調用R函數，還有就是要學會如何轉換常見的數據類型。

通常，可以將python的list對象，轉換成為R的vector對象 【robjects.ListVector()將python的字典轉換成R的列表】，之後直接使用R函數調用。 rpy2提供了幾個類，供我們把Python的list轉換成R的Vector。

相應的函數，如下：

robjects.StrVector()
robjects.IntVector()
robjects.FloatVector()
robjects.complexVector()
robjects.FactorVector()
robjects.BoolVector()
robjects.ListVector()

這些函數將python列表轉化成R的數據類型分別為：

  • 字串
  • 整數
  • 浮點數
  • 負數
  • 因子
  • 布林
  • 列表

# python 轉換成 R
# 使用vector系列函數時，輸入的只能是python的列表，而不能是數字或者字串

x = [1,2,3,4,5,6]
robjects.IntVector(x)

# R 轉換成 Python
# 推薦使用tuple()或者list()函數，將R對象轉換成tuple或者list。
a = r('c(1, 2, 3)')
# 此時為 r vector 型別 
# <class 'rpy2.robjects.vectors.FloatVector'>
print(type(a))

# 轉換成字串、tuple、list時已經變成 py型別
str(a)
tuple(a)
list(a)

第二種方式 - 引入 rpy2.ipython 物件¶

%%R -> 把一個 cell 都給 R %R -> 只是要執行一行, 就可以用 而 %R -o 可以把 R 的資訊傳給 Python, 反之用 %R -i 可以把 Python 的資訊傳給 R
